# Cybercrime Case Study – Phishing

Phishing is a very common type of cybercrime where attackers try to steal personal details like passwords, bank info, or OTPs by pretending to be someone trusted.

A normal situation can be like this — a student gets an email or message which looks like it’s from a bank or some popular app. The message usually creates panic, saying things like “your account will be blocked” or “urgent action required”. Without thinking much, the student clicks on the link given in the message.
These kinds of messages are very common nowadays, even I have seen similar emails in my inbox.

After clicking, it opens a website which looks almost real. Honestly, it’s very hard to tell the difference sometimes. The student then enters login details or bank information thinking it’s safe. But actually, all this data goes directly to the scammer.

These types of scams mostly target students and people who use the internet daily but don’t check everything carefully. In many cases, people lose money or their accounts get hacked, and recovering it is not easy.

This shows that we should not trust every message we receive. It’s always better to check properly before clicking any link, because one small mistake can create a big problem.
