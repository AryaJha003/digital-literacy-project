## About this Project
This repository contains my work for the Digital Literacy (CSE0001) course. In this project, I worked on different tasks related to digital tools, online platforms, communication, and cyber safety.

The main idea of this project was to understand how we use technology in our daily life and how to use it properly and safely.

## Tasks I Completed

- Digital Portfolio (GitHub, LinkedIn, Kaggle)  
- Coding practice and Google Form

- Cybercrime awareness
